### Changelog

#### WildFly-Camel 2.0.0.CR1

**Features**

* [#5][5] Provide camel-jta integration
* [#25][25] Provide camel-jaxb integration
* [#43][43] Provide camel-cxf soap integration
* [#44][44] Provide camel-cxf rest integration
